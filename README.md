# DevSpark

A self-hosted GitHub Spark alternative built with Deno, TypeScript, React, and SASS.

## Features

- Project tracking and management
- Code snippet sharing
- Collaboration features
- CLI interface
- Web interface

## Getting Started

### Prerequisites

- Deno 1.37 or higher
- Node.js 18 or higher (for client development)
